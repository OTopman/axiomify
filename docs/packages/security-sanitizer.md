# @axiomify/security-sanitizer

Sanitization primitives used by `@axiomify/security`.

## Install

```bash
npm install @axiomify/security-sanitizer
```

## Exports

- `sanitizeInput(input, options?)`
- `normalizeHpp(input)`

## Options

- `xssProtection`
- `prototypePollutionProtection`
- `nullByteProtection`
